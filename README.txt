# ChristmasGroup3Store
Online Christmas Selection Store



Index.html Landing page:
Animation Snow Fall and Current Christmas Countdown Clock based on user location.

User may enter in any search keyword and upon pressing enter on the keyboard a API pull from Walmart will bring up current items on next html page.

Search.html Second page:


Once user has entered a search term the second page is initialized and two API’s are pulled into action.

Walmart API and youtube API

Walmart API pulls the search information.
Youtube API pulls a view into Santa’s workshop.
